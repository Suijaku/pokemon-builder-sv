# Pokémon Builder SV

[https://suijaku.github.io/pokemon-builder-sv/](https://suijaku.github.io/pokemon-builder-sv/)

『ポケットモンスター スカーレット・バイオレット』向けのポケモン構築・管理ツールです。

## 概要
ブラウザ上でポケモンの個体情報（技、努力値、テラスタイプ等）を細かく設定し、パーティを構築・管理することができます。

## 主な目的
- **構築パーティの保存**: 自分が作成したパーティの構成をデータとして保存・管理します。
- **パーティの共有**: 作成したボックス情報をBase64形式のURLやGist連携を通じて、他のユーザーと簡単に共有することが可能です。

## 主な機能
- ポケモン・技・特性・アイテムの日本語検索・選択
- 実数値、努力値、個体値のシミュレーション
- ボックス登録とパーティ編成
- URLによるデータのエクスポート・インポート
